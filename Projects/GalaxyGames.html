Overview

Galaxy Games is a fast, clean, and constantly-updated unblocked games site built for students, schools, and anyone who needs safe access to browser-based HTML5 games. The hub runs entirely from local game files instead of embedding from external CDNs. Deploy it anywhere, customize it easily, and keep your access unblocked.

Website: galaxygames.site

Features

Games: 100% unblocked HTML5, Flash, and emulator games

Interface: Fast, lightweight, and desktop responsive

Network Access: Works on school networks

Deployment: One-click deploy to Netlify, Vercel, Cloudflare Pages, or GitHub Pages

Customization: Easily add new games and update the list

License: GPL-2.0 Open Source

Manual Deployment

If you donâ€™t want to use the Netlify button:

1. Fork or download the repository.
2. Upload it to any static host (Netlify, Vercel, Cloudflare Pages, GitHub Pages).
3. Wait for deployment to finish.
4. Your unblocked game site is live.

To add new games: place the game folder inside games-files/, add its icon inside Images/, and update your game list in games/.

Bypass Tips

Galaxy Games works best on domains that schools rarely block: Netlify subdomains, Vercel subdomains, Cloudflare Pages, GitHub Pages. Use multiple mirrors if needed to maximize uptime.
